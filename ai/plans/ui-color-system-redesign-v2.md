# UI 配色系统重设计 v2.0

## 设计理念

基于 ColorHunt 配色方案，打造温暖、自然、高对比度的双主题配色系统。

## 色彩方案

### 亮色主题 - "温暖米白 Warm Cream"

#### 背景层次
- **主背景**: `#FDF7F4` (米白) - 温暖柔和的基础色
- **卡片背景**: `#F9F6F3` (奶白) - 轻微对比的面板色
- **按钮背景**: `#F5C9B0` (桃粉米) - 柔和的交互元素
- **悬停背景**: `#EBE1D1` (米黄) - 温暖的悬停状态

#### 主题色（绿色系）
- **主绿色**: `#41644A` (森林绿) - 沉稳的主色调
- **柔和绿**: `#8EB486` (柔和绿) - 次要绿色
- **橄榄绿**: `#A6B28B` (橄榄绿) - 辅助绿色

#### 强调色（暖色系）
- **主橙色**: `#E9762B` (活力橙) - 强调和警告
- **珊瑚橙**: `#FD8A6B` (珊瑚橙) - 次要强调
- **桃色**: `#F5C9B0` (桃粉) - 柔和强调

#### 对比色（冷色系）
- **蓝灰色**: `#465C88` (蓝灰) - 信息和平衡
- **青绿色**: `#689B8A` (青绿) - 清新对比

#### 文字色
- **主文字**: `#000000` (纯黑) - 最高对比度
- **次文字**: `#685752` (深棕灰) - 柔和文字
- **弱文字**: `#997C70` (棕灰) - 辅助文字

### 暗色主题 - "深绿夜幕 Deep Forest Night"

## 色彩应用

### 聊天面板
- **用户头像**: 暖橙渐变 (`#E9762B` → `#FD8A6B`)
- **桌宠头像**: 青绿渐变 (`#689B8A` → `#465C88`)
- **用户消息**: 桃粉背景 (`#F5C9B0`)
- **桌宠消息**: 柔和绿背景 (`#8EB486`)

### 番茄钟面板
- **Focus 模式**: 森林绿 (`#41644A`)
- **Rest 模式**: 活力橙 (`#E9762B`)
- **完成徽章**: 青绿色 (`#689B8A`)

### 任务面板
- **高优先级**: 活力橙 (`#E9762B`)
- **中优先级**: 森林绿 (`#41644A`)
- **低优先级**: 青绿色 (`#689B8A`)

### 按钮变体
- **Primary**: 森林绿 (`#41644A`)
- **Success**: 柔和绿 (`#8EB486`)
- **Warning**: 活力橙 (`#E9762B`)
- **Info**: 蓝灰色 (`#465C88`)
- **Gradient**: 绿色渐变
- **Glass**: 玻璃态效果

## 色彩平衡

### 亮色主题
- **绿色系**: 40% (主色调)
- **暖色系**: 30% (强调色)
- **冷色系**: 15% (对比色)
- **中性色**: 15% (背景和文字)

### 暗色主题
- **绿色系**: 35% (主色调)
- **暖色系**: 35% (强调色)
- **冷色系**: 15% (对比色)
- **中性色**: 15% (背景和文字)

## 设计优势

1. **高对比度**: 所有文字和交互元素符合 WCAG AA/AAA 标准
2. **温暖自然**: 米白和森林绿营造温暖舒适的氛围
3. **活力强调**: 橙色系提供足够的视觉冲击力
4. **冷暖平衡**: 青绿和蓝灰提供视觉平衡
5. **层次清晰**: 背景层次分明，信息架构清晰
