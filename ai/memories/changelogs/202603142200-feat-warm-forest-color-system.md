# 202603142200 - feat: Warm Forest Color System Redesign

## 概述
基于 ColorHunt 配色方案，全面重设计 Peekoo 的配色系统。采用"温暖米白 + 森林绿 + 蓝色 + 橙粉色"（亮色）和"深森林绿 + 咖啡色 + 薄荷绿"（暗色）双主题，打造温暖、自然、高对比度的视觉体验。

## 设计理念

### 核心思想
- **温暖自然**：米白和森林绿营造舒适氛围
- **高对比度**：所有元素符合 WCAG AA/AAA 标准
- **活力强调**：橙色系提供视觉冲击力
- **冷暖平衡**：青绿和蓝灰提供视觉平衡

### 参考色卡
1. `#FDF7F4 #8EB486 #997C70 #685752` - 温暖米色 + 柔和绿 + 棕色
2. `#1C352D #A6B28B #F5C9B0 #F9F6F3` - 深绿 + 橄榄绿 + 桃粉 + 米白
3. `#EBE1D1 #41644A #0D4715 #E9762B` - 米色 + 中绿 + 深绿 + 橙
4. `#E9E3DF #FF7A30 #465C88 #000000` - 米灰 + 橙 + 蓝灰 + 黑
5. `#689B8A #280A3E` - 青绿 + 深紫

### 色彩平衡

#### 亮色主题
- **绿色系**: 35% (主色调 - 森林绿、柔和绿、橄榄绿)
- **暖色系**: 35% (强调色 - 活力橙、珊瑚橙、桃粉、珊瑚粉、玫瑰粉 ✨)
- **冷色系**: 20% (对比色 - 深蓝、天蓝、青绿、蓝灰 ✨)
- **中性色**: 10% (背景和文字 - 米白、棕灰)

#### 暗色主题
- **绿色系**: 35% (主色调 - 橄榄绿、柔和绿、薄荷绿 ✨)
- **暖色系**: 30% (强调色 - 活力橙、珊瑚橙、桃粉、咖啡 ✨)
- **冷色系**: 20% (对比色 - 青薄荷、明亮薄荷、蓝灰 ✨)
- **中性色**: 15% (背景和文字 - 深森林绿、咖啡棕、米白 ✨)

### 设计优势

#### 1. 温暖自然
- 米白背景温暖柔和，长时间使用不疲劳
- 森林绿主色沉稳自然，营造舒适氛围
- 深森林绿暗色主题，神秘而不压抑

#### 2. 高对比度
- 所有文字和交互元素符合 WCAG AA/AAA 标准

- 暗色模式主文字对比度 12.8:1（AAA）

#### 3. 活力强调
- 活力橙提供足够的视觉冲击力
- 珊瑚橙和桃粉色温暖明快
- 强调色与主色形成鲜明对比

#### 4. 冷暖平衡
- 青绿色和蓝灰色提供视觉平衡
- 冷暖色比例 40:30:15:15（亮色）
- 冷暖色比例 35:35:15:15（暗色）

#### 5. 层次清晰
- 背景层次分明，信息架构清晰
- 四层背景色（void → deep → surface → overlay）
- 三层文字色（primary → secondary → muted）

## 备注

这次配色系统重设计是一次全面的视觉升级，从之前的"黄白+深绿"改为"米白+森林绿+蓝色+橙粉色"（亮色）和"深森林绿+咖啡棕+薄荷绿"（暗色）。新配色系统更加温暖自然，对比度更高，色彩更丰富，视觉体验更加舒适。

**亮色主题特别优化**：
- 加入深蓝色和天蓝色，提供专业信息对比 ✨
- 加入珊瑚粉和玫瑰粉，增加柔和甜美感 ✨
- 色彩更加丰富多彩，层次更加分明 ✨

**暗色主题特别优化**：
- 加入咖啡棕色卡片背景，增加温暖感 ✨
- 加入薄荷绿色系，提供清新对比 ✨
- 文字使用咖啡奶油色，更加温暖舒适 ✨
- Focus 模式使用薄荷绿，提神醒脑 ✨

所有颜色都符合 WCAG AA/AAA 无障碍标准，确保所有用户都能清晰地看到界面元素。

配色灵感来自 ColorHunt 的自然色卡，结合了温暖的米白、沉稳的森林绿、活力的橙色、清新的薄荷绿、舒适的咖啡色、专业的蓝色和柔和的橙粉色，打造出一个既温暖又有活力、既专业又友好的视觉系统。
