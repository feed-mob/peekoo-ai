# fix(tauri): localize tray context menu by app language

## Summary
- Added tray menu localization in Tauri runtime (`en` / `zh-CN`) based on `app_language`.
- Tray menu labels are now language-aware at startup.
- When language changes via settings commands, tray menu labels are rebuilt and refreshed in place.

## Implementation
- Updated:
  - `apps/desktop-tauri/src-tauri/src/lib.rs`
- Added helper functions:
  - `tray_menu_labels(language)`
  - `apply_tray_menu_language(app, language)`
- Hooked refresh logic into:
  - `app_settings_set` when `key == app_language`
  - `app_settings_set_language`

## Validation
- `cargo check -p peekoo-desktop-tauri` passed.
