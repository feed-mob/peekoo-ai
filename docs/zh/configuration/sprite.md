# Sprite 配置

Sprite 是 Peekoo 常驻在桌面上的角色形象。你可以使用内置的角色，也可以上传自己的图片来定制专属桌宠。

<!-- 图片占位：Sprite 切换界面截图 -->

## 切换内置 Sprite

系统托盘图标右键 → 设置 → 外观 → 精灵形象

目前内置了两个角色：

- **Dark Cat** — 默认深色主题猫咪
- **Cute Dog** — 可爱风格小狗

选择后即时生效。

## 自定义 Sprite

如果你想上传自己的角色形象，Peekoo 支持通过设置页面完成整个流程。

系统托盘图标右键 → 设置 → 外观 → 自定义精灵

有两种方式准备图片：

### 方式一：用 AI 生成精灵图

1. 点击「复制提示词」
2. 把提示词粘贴到你常用的 AI 绘图工具（如 ChatGPT、Midjourney 等）
3. 在提示词末尾填入角色名称、风格和形象描述
4. 生成并下载图片

### 方式二：自己准备图片

如果你已经有现成的图片，或者想自己绘制，按照下方的格式要求制作好精灵图后直接上传即可。

### 上传图片

准备好图片后：

1. 点击「上传图片」选择精灵图文件
2. Peekoo 会自动尝试生成 manifest 配置
3. 点击「校验草稿」检查是否有问题
4. 确认无误后点击「保存自定义精灵」

也可以点击「上传 manifest」手动提供配置文件，或直接在 JSON 编辑器中调整参数。

## 精灵图格式要求

Peekoo 使用精灵图（sprite sheet）格式，即把所有动画帧排列在一张图片上。

### 基本规格

- 布局：**8 列 × 7 行**，共 56 帧，每个帧区域必须是正方形
- 整体长宽比：约 8:7
- 推荐分辨率：**4096 × 3584**（每帧 512 × 512）；程序会自动缩放到 1024 × 896
- 格式：PNG（推荐）或 JPG

### 背景色要求

- 背景色必须且只能是**纯品红 `#ff00ff`**
- 不要使用渐变、阴影、纹理或噪点
- 角色本身不要包含接近品红的颜色（阴影、反光、轮廓光等），否则会被一起抠掉
- 帧与帧之间、图片四周都必须保持纯品红，不要出现黑缝或深色分隔线

### 帧布局要求

- 不要绘制任何网格线、边框或分隔线，帧与帧之间只通过位置区分
- 每帧内部保留少量安全边距，确保角色不会被裁切
- 角色在每一帧中尽量居中对齐，位置保持一致

### 动画连贯性

- 同一行为同一个动画，从左到右依次播放，循环
- 相邻帧之间必须连贯，不要跳帧或突然大幅位移
- 第 1 行（待机）需要有轻微的呼吸和眨眼动作，不要完全静止
- 第 6 行（休息）只画闭眼呼吸，不要画打哈欠

### 各行动画说明

| 行 | 动画 | 触发时机 | 说明 |
|----|------|---------|------|
| 1 | 待机 / 偷看 | 默认状态 | 轻柔呼吸，从屏幕底端露出半个身体偷看，偶尔眨眼 |
| 2 | 开心 / 庆祝 | 完成任务 | 欢快表情和庆祝动作 |
| 3 | 工作 / 专注 | 番茄钟进行中 | 专注表情和工作姿态 |
| 4 | 思考中 | AI 处理请求 | 思考表情和姿态 |
| 5 | 提醒 / 通知 | 任务截止、插件通知 | 提醒表情和动作 |
| 6 | 困倦 / 休息 | 休息状态 | 疲倦表情，闭眼呼吸 |
| 7 | 拖拽移动 | 被拖动时 | 惊讶或配合移动的表情 |

## manifest.json 字段说明

如果你需要手动调整配置，以下是主要字段：

```json
{
  "id": "my-sprite",
  "name": "My Sprite",
  "description": "描述",
  "image": "sprite.png",
  "layout": {
    "columns": 8,
    "rows": 7
  },
  "scale": 0.40,
  "frameRate": 6,
  "chromaKey": {
    "targetColor": [255, 0, 255],
    "threshold": 100,
    "softness": 80,
    "spillSuppression": {
      "enabled": true,
      "threshold": 260,
      "strength": 0.90
    },
    "pixelArt": false
  }
}
```

| 字段 | 说明 |
|------|------|
| `scale` | 显示缩放比例，影响桌面上的角色大小 |
| `frameRate` | 动画帧率，建议 5–8 |
| `chromaKey.targetColor` | 背景色，固定为品红 `[255, 0, 255]` |
| `chromaKey.threshold` | 抠图阈值，值越大抠得越多 |
| `chromaKey.softness` | 边缘柔化程度 |
| `chromaKey.spillSuppression` | 抑制角色边缘的品红色溢出 |
| `pixelArt` | 像素风格模式，像素画角色建议开启 |

## 实用建议

- 帧率建议从 6 开始调整，太快会显得焦躁，太慢会显得卡顿
- 在你常用的桌面背景颜色上测试一次抠图效果
- 按 Peekoo 实际桌面显示尺寸检查效果，不要只看原图
