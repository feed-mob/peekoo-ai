# Plugins

## 概览

Peekoo 支持基于 WASM 的沙箱插件。插件可以增加：

- Agent 工具
- UI 面板
- 事件钩子
- 持久化插件状态

插件是 Peekoo 超出内置聊天、任务和番茄钟能力之后，继续扩展功能的主要方式。它让 Peekoo 从一个聚焦的桌面伙伴，逐步成长为更灵活、更可定制的工作台。

## 发现位置

插件会从以下位置被发现：

- `~/.peekoo/plugins/`
- 工作区本地的 `plugins/`

## 运行模型

插件通过插件宿主和 Extism 运行时加载。UI 面板运行在受限 webview 中，工具也可以反向暴露给 Agent 运行时使用。

## 插件的常见形态

不同插件带来的体验可能不一样：

- 纯工具型扩展，为 Agent 增加新能力
- 面板型集成，在 Peekoo 内加入专门的 UI
- 事件驱动型工具，在运行时事件发生时做出响应

## 如何选择第一个插件

建议先从一个能解决明确需求的插件开始。最适合的通常是那些可以减少重复手工步骤的集成，例如接入外部服务数据，或者提供一个专用工作流面板。
