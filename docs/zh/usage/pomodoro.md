# Pomodoro

## 概览

Peekoo 内置了番茄钟计时器，用于专注工作。

它被设计成和产品其他能力紧密配合。你可以从记录任务开始，进入一次专注会话，再在结束时补一条简短备注，而且整个过程都不需要离开同一个伙伴。

## 核心操作

当前运行时支持：

- 启动专注或休息会话
- 暂停当前会话
- 恢复已暂停的会话
- 结束或取消会话
- 切换模式
- 查看会话历史
- 为番茄钟会话保存备注
- 配置番茄钟设置

## 典型专注流程

1. 先选定你要专注的任务。
2. 启动一次专注会话。
3. 只有在确实需要离开时才暂停。
4. 当这一轮工作结束后完成会话。
5. 如果你想保留记录，可以补一条备注。

这套流程很适合个人规划，因为计时器、任务系统和历史记录都在同一个应用里。

## 与任务关联

在保存专注备注时，番茄钟历史可以附带一个可选任务关联。这样可以把专注记录和任务关联起来，同时避免通过副作用去修改任务评论。

## 适合什么时候使用

当你希望做到以下几件事时，Pomodoro 会很有帮助：

- 保护一小段不被打断的专注时间
- 围绕一个明确任务建立推进节奏
- 为完成内容留下轻量记录
- 之后回看最近的专注历史

## 为什么是内置能力

番茄钟与任务、设置同属一套生产力能力，因此它既能通过 UI 使用，也能通过 MCP 暴露给 Agent 调用。
