# 番茄钟

Peekoo 内置了番茄钟计时器，帮助你保护专注时间、建立工作节奏。计时器、任务系统和历史记录都在同一个应用里，不需要切换工具。

<!-- 图片占位：番茄钟面板截图 -->

## 打开番茄钟

右键 Sprite → 番茄钟

也可以直接在聊天中让 Agent 帮你启动，例如：

> Peekoo，帮我开启一个 25 分钟的专注番茄钟

## 基本操作

| 操作 | 说明 |
|------|------|
| 开始 | 启动当前模式的计时 |
| 暂停 / 继续 | 暂时中断或恢复计时 |
| 重置 | 结束当前会话，回到初始状态 |
| 切换模式 | 在专注和休息之间手动切换 |

计时器下方是每日徽章墙，用小图标记录当天所有的专注和休息次数。

## 典型专注流程

1. 打开番茄钟面板
2. 点击「开始」启动专注计时
3. 专注时间结束后，切换到休息模式
4. 休息结束后开始下一轮
5. 会话结束后可以写一条备注，记录完成了什么，并关联对应的任务

## 专注备注

每次会话结束后，可以写一条简短备注，备注窗口与任务系统联动，可以直接选择对应的任务关联：

- 备注内容：记录这段时间完成了什么、收获、心得或灵感
- 关联任务：从未完成的任务列表中选择，把专注记录和任务绑定在一起

如果开启了「专注备注」选项，会话结束时会自动弹出备注窗口；也可以在历史记录中展开任意条目随时补写。

## 历史记录

面板下方显示最近的会话记录，可以按时间范围筛选：

- 最近 6 条
- 今天
- 昨天
- 最近 7 天
- 最近 30 天

每条记录显示模式（专注 / 休息）、结束时间、实际用时和计划时长，以及完成或中断状态。

## 设置

点击面板右上角的设置图标进入：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 专注时长 | 每轮专注的分钟数 | 25 分钟 |
| 休息时长 | 短休息的分钟数 | 5 分钟 |
| 长休息时长 | 长休息的分钟数 | 15 分钟 |
| 周期 | 几轮专注后触发长休息 | 4 轮 |
| 专注备注 | 会话结束后自动弹出备注窗口 | 关闭 |
| 自动模式 | 会话结束后自动进入下一阶段 | 关闭 |

修改后点击「保存更改」生效。
