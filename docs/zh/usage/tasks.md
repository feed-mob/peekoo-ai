# Tasks

## 概览

Peekoo 的 Tasks 属于内置生产力系统的一部分。它给你一个轻量、始终可见的位置来记录工作、追踪进度，并把这些内容和应用的其他能力连接起来。

任务能力由 React UI、Agent 工具和插件集成共同复用。

## 当前支持的能力

当前代码库中的任务能力包括：

- 创建任务
- 列出任务
- 更新任务
- 删除任务
- 切换完成状态
- 分配任务
- 添加评论
- 更新标签和状态

## 为什么重要

同一套任务服务可同时被以下入口使用：

- 桌面 UI
- 内置 Agent
- 插件

这样可以让人工操作和 Agent 驱动的工作流共享一致的数据。
