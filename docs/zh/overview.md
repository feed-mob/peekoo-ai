# 产品概览

Peekoo AI 是一个直接生活在桌面上的 AI 伙伴。它把轻量级桌面宠物、聊天界面、生产力工具和扩展系统组合在一起，让你在不离开当前工作流的前提下完成更多事情。

Peekoo 的目标，是让 AI 支持变得更贴近日常工作，而不是每次都要切换到浏览器标签页或一个完整的大窗口里。它保持在桌面边缘，随时可用，既能陪伴，也能帮助你思考、规划和保持专注。

## 核心能力

- 支持流式回复和可配置提供商的 AI 聊天
- 在桌面陪伴形态下完成任务管理
- 支持暂停、恢复、结束和历史记录的番茄钟
- 具备情绪反馈与桌面存在感的动画 Sprite 角色
- 通过插件扩展工具和面板

## 架构概览

Peekoo 使用分层的桌面应用架构：

```text
desktop-ui -> desktop-tauri -> peekoo-agent-app -> domain/runtime crates
```

`desktop-ui` 负责 React 体验层。`desktop-tauri` 是传输层。`peekoo-agent-app` 负责任务、番茄钟、设置、Agent 运行时和插件集成的编排。

## 为什么 Peekoo 的体验不一样

Peekoo 的设计重点不是“打断你”，而是“陪伴你”。Sprite 让应用在桌面上始终有一个可见的位置，聊天界面让 AI 触手可及，内置生产力工具则减少了来回切换上下文的成本。你可以在同一个伙伴中完成提问、记录任务和启动专注流程。

## 面向扩展而设计

Peekoo 不只包含内置能力。通过插件系统，它可以逐步加入新的工具、面板和第三方集成。因此，它既适合个人日常使用，也适合逐渐演化出更定制化的工作流。
