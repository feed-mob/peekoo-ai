# 产品概览

> Peekoo: A little peek, a lot of help.

Peekoo 是一个以桌宠陪伴为核心、通过插件系统无限扩展的 AI 智能体应用。

大多数时候，它只是悄悄趴在桌面的角落里偷看你，时不时打个瞌睡——就像每一个桌宠该做的那样。但在你需要的时候，它随时准备好成为你的帮手。

<!-- 图片占位：Peekoo 桌面截图 -->

我们把 Peekoo 定义为**可无限扩展、量身定制的桌面小精灵**。这个项目起源于一个问题的思考：AI 到底应该如何以更温和的方式融入工作和学习？

Peekoo 的答案是：不打扰，但随时在。

## 核心能力

- **AI 聊天** — 随时开启对话，支持多个 AI 提供商，可连接 OpenCode 等 CLI 工具，一键集成 AI Agent，无需代码背景也能轻松配置
- **任务管理** — 原生内置任务系统，在桌面伙伴旁边创建和跟踪任务
- **番茄钟** — 内置专注计时器，支持暂停、恢复、结束和历史记录
- **Sprite 角色** — 支持自定义桌宠形象，可按格式上传图片，也可使用内置的图片 Prompt 生成
- **Skill 配置** — 上传 Skill 文件扩展 Agent 能力，后续将支持 MCP 配置
- **插件商店** — 持续上新，可无限扩展 Peekoo 的能力边界

## 已支持的插件

| 插件 | 功能 |
|------|------|
| 健康提醒 | 工作间隙提醒你喝水、护眼、站起来活动 |
| OpenClaw Sessions | 直接通过 Peekoo 打开 OpenClaw 的 Session 管理窗口 |
| Claude Code / OpenCode Companion | Agent 的每一次思考和输出，都与 Peekoo 的表情同步 |
| Mijia Smart Home | 用 Peekoo 管理你的米家智能家居设备 |
| Google Calendar | 导入日程，通过聊天直接进入 Google Meet |
| Linear | 导入已有的 Todolist，与 Peekoo 任务系统同步 |

……以及更多持续上新中。

## 设计理念

Peekoo 的核心设计理念是将"基础陪伴"与"定制需求"合理分离，避免功能堆砌带来的冗杂感；同时将 AI Agent 的工作过程可视化和人格化，为冷硬的工具赋予温度和反馈。

我们期待凭借无限的插件去扩建 Peekoo 的神经网络——它将拥有无限的可能性。

## 支持平台

Windows、macOS、Linux，均提供安装包下载。

- 需要轻量任务管理和专注计时的个人用户
