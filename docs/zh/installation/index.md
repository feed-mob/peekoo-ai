# 安装总览

Peekoo 通过 GitHub Releases 提供桌面版本，因此你可以直接下载安装，而不需要先从源码构建。

## 支持的平台

- Windows：安装包
- macOS：Apple Silicon DMG
- Linux：AppImage 和 DEB，另提供 Arch Linux AUR 包

请选择最适合你当前平台和安装习惯的方式。

## 平台说明

- [macOS](./macos.md)
- [Windows](./windows.md)
- [Linux](./linux.md)

## 说明

- macOS 尚未完成 notarization，安装后可能需要手动移除 quarantine 属性。
- Windows 尚未配置代码签名，因此仍可能看到 SmartScreen 警告。

这些提示主要与打包分发方式有关，不影响 Peekoo 的核心使用路径。完成安装并允许应用启动后，日常体验是一致的。
