# Install on Windows

On Windows, Peekoo is distributed as a standard installer so setup stays familiar and easy to follow.

## Download

1. Open the [latest GitHub Release](https://github.com/feed-mob/peekoo-ai/releases/latest).
2. Download the `x64-setup.exe` installer.
3. Run the installer.

## Notes

- Windows code signing is not configured yet.
- SmartScreen may warn before launch.

If Windows blocks the app, review the publisher warning carefully and continue only if you trust the downloaded release.
