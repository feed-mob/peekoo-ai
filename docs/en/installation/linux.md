# Install on Linux

Peekoo supports common Linux installation paths, from direct release downloads to AUR installation on Arch Linux.

## Release Builds

Peekoo publishes Linux artifacts in GitHub Releases:

- AppImage
- DEB

Download the format that matches your environment.

## Arch Linux

Install from the AUR:

```bash
yay -S peekoo-bin
```

Package: [peekoo-bin](https://aur.archlinux.org/packages/peekoo-bin)
