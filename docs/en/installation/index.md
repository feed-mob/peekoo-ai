# Installation Overview

Peekoo ships desktop builds through GitHub Releases, so you can get started without building from source.

## Supported Targets

- Windows: installer builds
- macOS: Apple Silicon DMG
- Linux: AppImage and DEB, plus Arch Linux AUR packaging

Choose the path that matches your platform and setup style best.

## Platform Guides

- [macOS](./macos.md)
- [Windows](./windows.md)
- [Linux](./linux.md)

## Notes

- macOS notarization is not configured yet. You may need to remove the quarantine attribute after install.
- Windows code signing is not configured yet. SmartScreen warnings may still appear.

These prompts are packaging-related rather than product-related. Once the app is installed and allowed to launch, the everyday Peekoo experience is the same.
